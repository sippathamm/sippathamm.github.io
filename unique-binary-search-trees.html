Unique Binary Search Trees (Dynamic Programming)

By Sippawit Thammawiset

November 9, 2021
  • Share this:

Unique Binary Search Tree อธิบายปัญหาสั้น ๆ ได้ว่า กำหนดให้จำนวนเต็ม n คืนค่าจำนวนต้นไม้ที่ไม่ซ้ำกัน (Binary Search Tree) ซึ่งมี n โหนดที่มีค่าตั้งแต่ 1 ถึง n จากรูปที่ 1 เมื่อ n = 3 จะสามารถสร้าง Binary Search Tree ได้ทั้งหมด 5 แบบ โดย Binary Search Tree ที่ 1 กับ 5 (จากซ้ายไปขวา) ถือว่าเป็นคนละแบบ ในขณะเดียวกัน Binary Search Tree ที่ 2 กับ 3 ก็ถือว่าเป็นคนละแบบเช่นกัน

แน่นอนว่าเมื่อเจอ Binary Search Tree จะต้องมีเรื่องของ Recursion เข้ามาเกี่ยวข้อง จากรูปที่ 2 หากลองสังเกตดูดี ๆ BST (ขอย่อ Binary Search Tree ว่า BST) ที่มี 3 Node มันสร้างมาจาก BST ที่มี 2 Node และ 1 Node (Sub-tree) เพียงแค่สลับตำแหน่งกันไปมาเท่านั้นเอง กำหนดให้ c(n) แทนจำนวนวิธีที่สามารถสร้าง BST ที่มี n Node และมี Base case คือ เมื่อ n <= 1 ก็ return 1 ไปซะ (BST ที่มี 1 Node สร้างได้แค่แบบเดียว)

แนวคิดต่อมาคือ เราจะให้หนึ่ง Node เป็นรากอยู่กับที่ หาก BST นี้มี n Node เราจะได้ว่า Sub-tree ทางซ้ายมี n – n Node นั่นคือสามารถสร้าง BST ทางซ้ายได้ c(n – n) วิธี ในขณะเดียวกัน Sub-tree ทางขวามี n – 1 Node นั่นคือสามารถสร้าง BST ทางขวาได้ c(n – 1) วิธี เมื่อรวมกันก็ได้เป็น BST ต้นใหญ่ที่มี n Node (0 Node ทางซ้าย + 1 Node ที่เป็นราก + n – 1 Node ทางขวา) ซึ่ง Sub-tree ทางซ้ายและ Sub-tree ทางขวาของต้นไม้นี้เป็นอิสระต่อกัน จึงเป็นการคูณไม่ใช่การบวก และก็หาผลรวมทุกกรณีที่ Node ของ Sub-tree ทางซ้ายและ Sub-tree ทางขวารวมกันได้ n Node

ดูแล้วก็คุ้น ๆ เหมือนทฤษฎีบททวินามเลยแฮะ แต่มันคือ Catalan number มีประโยชน์ในเรื่องของการหาจำนวนวิธีที่สามารถสร้างวงเล็บ n คู่ โดยที่วงเล็บทั้งหมดนั้นต้องเรียงกันอย่างถูกต้อง เช่น n = 3 สามารถสร้างได้ 5 แบบ ดังนี้ ((())), ()(()), ()()(), (())() และ (()()), การหาจำนวนวิธีที่สามารถสร้าง BST ที่มี n Node (บทความนี้นี่เอง), การหาจำนวนวิธีการเดินทางจากมุมซ้ายล่างไปยังมุมขวาบนของรูปสี่เหลี่ยม n x n โดยที่สามารถเดินได้เพียง 2 แบบคือขึ้นและไปทางขวาและห้ามผ่านเส้นทแยงมุมของรูปสี่เหลี่ยม เป็นต้น ซึ่ง Sequence ของ Catalan number นั้นเติบโตแบบน่ากลัวมาก (1, 1, 2, 5, 14, 42, 132, 429, 1430, 4862, …) แถม Recurrence ของมันก็ยังเกิดการทำงานซ้ำซากอีก เลยต้องใช้ Dynamic Programming และ Memoization เข้ามาช่วยจดจำคำตอบ
